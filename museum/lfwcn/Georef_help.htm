Map Help

Important notes:
The refresh and back/forward buttons will not behave as expected so please avoid using them. Pressing refresh will return the map to the state at which it was when you first entered the page. Pressing back/forward will not take you to the previous or next state of the map, but rather navigate to the previous or next page you visited outside of this one. Any map state can be achieved using the map controls.

Whenever the 'HUCS' or 'Eco-regions' layer is selected, it becomes the background, so it automatically overrides the 'Counties' layer.
Throughout this help page the large map with the controls around it will be referred to as the primary map and the smaller map next to the primary one will be referred to as the reference map.

Clicking on both maps at the maximum scale will zoom in to the point clicked, while clicking on them at any other scale will simply pan the viewing area to that point at the same scale. The reference map allows you to navigate the whole map, even to the areas not currently shown in the primary map's viewing area.
Resets the map to the full scale.
Zooms the map in one scale. If the map's top layer's minimum scale has already been reached, zooming in will automatically display the next layer that supports the requested scale. Simply zoom out to return to the previous layer.
Zooms the map out one scale.
Displays this help page.

The pan controls will move the current viewing area almost a full length in the desired direction.
To select a custom viewing area left-click, hold and drag the mouse on the primary map in any direction. A dynamic box similar to this one will appear and the map will zoom in to it's approximate dimensions upon releasing the mouse button.
